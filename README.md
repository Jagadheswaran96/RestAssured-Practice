# Rest Assured Practice 🚀 

A practice project showcasing **API automation testing** using [Rest Assured](https://rest-assured.io/), TestNG, and supporting libraries. 

This repository is designed to help SDETs and QA engineers quickly set up a working framework for REST API testing. 

--- 

## 📌 Features 

- API testing with **Rest Assured 5.4.0**
- JSON & XML validation
- POJO serialization/deserialization with **Jackson**
- Logging support via **SLF4J**
- Test execution with **TestNG**
            
--- ## 🛠️ Tech Stack & Dependencies

- `io.rest-assured:xml-path:5.4.0`

- `org.apache.groovy:groovy-json:4.0.21`
